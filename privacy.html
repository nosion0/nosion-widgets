개인정보 처리방침

시행일: 2026-06-05 · 최종 개정: 2026-06-05

노시언 위젯(이하 "서비스")은 노션(Notion) 임베드용 미니멀 HTML 위젯 모음입니다. 본 방침은 서비스 이용 과정에서 수집·처리되는 정보와 그 보호 방법을 안내합니다.

1. 데이터 처리 원칙

  • 저장 위치: 모든 위젯 설정·데이터는 사용자 브라우저의 localStorage에만 저장되며, 자체 서버를 운영하지 않습니다.
  • 회원가입·로그인 없음: 별도 계정 시스템이 없으며, 이메일·전화번호·이름 등의 신원 정보를 수집하지 않습니다.
  • 광고·추적 없음: 광고 식별자, 분석 스크립트, 추적 쿠키를 사용하지 않습니다.

2. 처리 정보 및 외부 송신

서비스는 위젯 기능 제공을 위해 다음의 외부 API를 호출하며, 이때 일부 정보가 해당 제공자에게 전송될 수 있습니다.

2.1. IP 주소에 관하여

달 위상·별자리·일출일몰·공기질 위젯이 "현재 위치" 자동 감지를 시도할 때 사용자 IP가 ipapi.co로 전송되며, 그 결과 대략적인 도시 단위 위치만 사용합니다. 사용자가 직접 도시를 선택하면 IP는 더 이상 전송되지 않습니다. 결과는 12시간 동안 브라우저에 캐시되어 요청 빈도를 최소화합니다.

2.2. CORS 프록시 안내 (주식 위젯)

주식 위젯은 Yahoo Finance 데이터를 받기 위해 외부 CORS 프록시(codetabs.com, allorigins.win, r.jina.ai)를 경유합니다. 전송되는 정보는 티커 기호뿐이며 개인 식별 정보는 포함되지 않으나, 해당 프록시 운영사의 정책에 의해 IP가 로깅될 수 있습니다.

3. 보관 및 삭제

  • 위젯 데이터는 사용자 브라우저에 저장되며, 서비스가 별도로 수집·보관하지 않습니다.
  • 외부 API 제공자의 보관 정책은 각 제공자의 약관을 따릅니다.
  • 브라우저의 사이트 데이터 삭제(쿠키 및 사이트 데이터 삭제) 또는 시크릿/프라이빗 창 사용으로 데이터를 즉시 제거할 수 있습니다.

4. 사용자 권리

  • 저장된 데이터에 대한 열람·정정·삭제 권리를 가집니다(브라우저 개발자 도구의 Application → Local Storage에서 확인 가능).
  • 일부 위젯(달 위상·별자리·일출일몰·공기질)은 사용자가 도시를 직접 선택해 IP 자동 감지 없이 사용할 수 있습니다.
  • 주식 위젯을 사용하지 않으면 CORS 프록시 의존 우려를 회피할 수 있습니다.

5. 미성년자

서비스는 만 14세 미만 아동을 표적으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다. 보호자의 안내하에 이용을 권장합니다.

6. 보안

  • 서비스는 자체 서버 없이 정적 호스팅(예: GitHub Pages)으로 배포되며, 모든 외부 통신은 HTTPS로 이루어집니다.
  • 외부 의존성(CDN, API) 침해로 인한 위험을 완전히 배제할 수 없으며, 합리적 수준의 방어 코딩(입력 검증, 출력 이스케이프 등)을 적용합니다.

7. 정책 변경

본 처리방침이 변경되는 경우 본 페이지 상단의 "최종 개정일"이 갱신되며, 중요한 변경 시 별도 안내를 제공합니다.

8. 문의

처리방침에 관한 문의는 서비스 운영자에게 GitHub Issue를 통해 전달해 주시기 바랍니다.

본 처리방침은 일반적인 안내이며, 법률 자문을 대체하지 않습니다. 사업적 활용 시 변호사의 검토를 받으시기 바랍니다.
